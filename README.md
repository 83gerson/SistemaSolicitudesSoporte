# Sistema de Registro de Solicitudes de Soporte Técnico

## Descripción breve
Aplicación web desarrollada en ASP.NET MVC (.NET Framework) que permite registrar, listar, consultar el detalle y filtrar solicitudes de soporte técnico, aplicando el patrón MVC, validaciones, JavaScript, AJAX y gestión de estado mediante Session y TempData.

## Tecnologías utilizadas
- ASP.NET MVC 5 (.NET Framework)
- C#
- Razor (vistas .cshtml)
- JavaScript / jQuery
- AJAX (jQuery $.ajax)
- Bootstrap
- Session / TempData

## Instrucciones básicas para ejecutar el proyecto
1. Clonar o descargar el repositorio.
2. Abrir el archivo `.sln` con Visual Studio.
3. Compilar la solución (Build > Build Solution).
4. Ejecutar con F5.
5. Ingresar un nombre en la pantalla de inicio para comenzar a usar la aplicación.
